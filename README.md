# 🔧 Technichno — Сервис по ремонту телефонов

**Веб-приложение для управления заказами в сервисном центре по ремонту мобильных телефонов**

---

## 🎯 О проекте

Система предназначена для автоматизации работы сервисного центра по ремонту телефонов. Позволяет принимать заказы, отслеживать их статус, управлять клиентами, финансами и отчётностью.

---

## ✨ Основные возможности

- **Управление заказами** (приём, обработка, выдача)
- **Учёт клиентов** и история ремонтов
- **Каталог услуг** и ценообразование
- **Финансовый модуль** (доходы, расходы, отчёты)
- **Админ-панель** для управления
- **Шаблоны документов** (квитанции, акты)
- Разделение по ролям (администратор, мастер, менеджер)

---

## 🛠 Технологии

- **Backend**: Python + Django
- **База данных**: SQLite / PostgreSQL
- **Frontend**: HTML + CSS + JavaScript (Bootstrap)
- **Управление зависимостями**: virtual environment
- **Структура**: Модульная архитектура (accounts, clients, orders, finance и т.д.)

---

## 🚀 Как запустить проект

```bash
# 1. Клонировать репозиторий
git clone https://github.com/dimakim0906-lgtm/technichno.git
cd technichno

# 2. Создать и активировать виртуальное окружение
python -m venv venv
venv\Scripts\activate    # Windows
# source venv/bin/activate  # macOS / Linux

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Применить миграции
python manage.py migrate

# 5. Создать суперпользователя (админа)
python manage.py createsuperuser

# 6. Запустить сервер
python manage.py runserver
